Види селекторів:

( * ) - Універсальний селектор - один стиль для всіх елементів сторінки (шрифт, підкреслення і т.п)

( # ) - вибирає всі елементи з заданою назвою id

( . ) - Вибирає элементи з заданим класом

( A > P) - Вибирає всі дочірні элементы A, у яких батьківський элемент P

( H2 + P ) вибирає всі елементи P,

які прозміщені одразу після H2

( H3 ~ p ) - Вибирає всі елементи Р, которі слідують одразу за элементом H3

( [=] ) - обирає вcі элементи з завданим атрибутом TYPE і класом CHECKBOX
( [] ) - обирає вcі элементи з атрибутом CLASS
